# 📚 CodeNotes-Library

A curated collection of **programming language PDF notes** designed for students, beginners, and developers.

This repository provides easy access to structured learning materials for multiple programming languages in one place.

---

# 🎯 Purpose

The goal of this repository is to create a **centralized library of programming notes** so learners can quickly download and study different programming languages.

It helps with:

📖 Quick revision  
📚 Learning new programming languages  
💻 Coding interview preparation  
🎓 Academic studies  

---

| Language | Description |
